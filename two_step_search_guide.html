Troubleshooting

If "Available Flights" shows "No Search Results":

  • Make sure you clicked the "Search" button first
  • Check that FROM, TO, and DATE are filled in
  • Wait for "Search Complete" message before clicking Available Flights

If Search returns no flights:

  • Add flights to database using the link above
  • Make sure XAMPP Apache & MySQL are running
  • Try a different route (e.g., Delhi ‚Üí Ahmedabad)
